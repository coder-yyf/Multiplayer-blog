# 仿[cnodejs.org](cnodejs.org)论坛官网

### 启动方法

1. 安装依赖包`$ npm install`
2. 开启数据库
   - `$ mongod`
   - `$ mongo`
3. 使用 nodemon 或 node,启动 app.js
   - `$ nodemon app.js` / `node app.js`
4. 在浏览器输入 `localhost:3000` 即可访问项目

### 技术栈

1. node.js
2. express
3. mongoDB
4. art-template
5. bootstrap
6. jquery

技术用的都比较简单，主要是通过这个项目对Express进行练手
